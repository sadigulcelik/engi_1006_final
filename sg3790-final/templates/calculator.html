Try something like '3^1.5+(4!-31)*2

Note that the calculator currently does not understand '-5' but will understand '(0-5)'

A stack based evaluator mechanism that currently supports addition, subtraction, multiplication, division, exponentials (I use the carrot convention) and factorials. Once the algorithm has been written out it is easier to add new function and customize it to support more operations one might use (such adding support for integration and differentiation).
