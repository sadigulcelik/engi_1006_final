Try something like 'x^2+x^1.2-3/x'

The grapher does not currently support something of the form -x^2, so instead just type '0-x^2'

I had initially struggled to find a way to get the user to input code that I would use to graph. Then, I realized that I already had a tool to do exactly what I wanted. I use the stack based algorithm (and nothing else) to evaluate any polynomial of x including some fractional powers and even (very roughly) factorials. Effectively, the code works by going through every x value, replacing the character 'x' with the value of x at that point, and computing the y value using my stack based evaluator.
